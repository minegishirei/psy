## 心理的安全性の欠如

心理的安全性が欠如した状態とは、簡単に言えば処罰を恐れる心理が萬栄下状態のこと。
特定のグループの中でリスクをとったり、間違えた行動が全く許容されない環境のことであり、恐怖の文化は透明度を下げ、必要な情報の伝達を阻害する。

1. 心理的安全性が欠如している状況
2. 情報の隠ぺいを引き起こし
3. 適切な質問や情報交換を阻害してしまうため、知識の共有という点でもよくない。
