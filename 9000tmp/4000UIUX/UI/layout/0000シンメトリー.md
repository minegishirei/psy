


# 非日常を取り入れたい 

<small>シンメトリー</small> 

<img width="80%" src="https://traveldiary.tokyo/wp-content/uploads/2017/05/20170511_09.jpg"> 

<p>自然界のものは基本的に不規則に配置されているはずだが
そこを逆手に取り、左右対称にものを置くことで非日常を演出する。
シンメトリーは神聖な空気を出しやすい。</p>
